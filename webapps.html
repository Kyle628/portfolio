Quilt

Quilt is a social imaging platform, built to enable people to work together creatively to create all new kinds of artwork. Whether it be a mashup of favorite memes, or some well placed cat pics, everyone can make a piece of the quilt their own. This web application was made by myself, and 4 others, using the web framework web2py. Check out the features menu to see some demos of the site in action.

Graphics

This is a simple graphics rendering program that I made using WebGL. Any model that has a vertex/polygon representation can be rendered. Once rendered, you can use your mouse to move/rotate the object. There are also toggles for flat/Gouraud shading, and orthographic perspective. Check out the features menu to see some demos of the program in action.

ShredJournal is a surf journaling web application. Shredjournal can be used as a simple way to remember your best sessions, or as a surf prediction tool, so that you know where to be when the swell hits.
